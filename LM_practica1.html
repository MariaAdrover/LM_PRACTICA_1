Mi primera página web

Esta página es el primer ejercicio práctico del módulo Llenguatge de marques (Tècnic superior en Desenvolupament d’aplicacions multiplataforma). En ella comentaré de forma muy básica algunas de las cosas que he aprendido y que he tenido en cuenta para elaborar mi primera página web.

La página propuesta para el ejercicio ha de tener estos elementos:

  1. Título y descripción en el head
  2. Título informativo en h1
  3. Una o más imágenes redimensionadas, con el atributo alt que contenga entre 4 y 8 palabras que describan la imagen
  4. Una tabla de datos relacionada con el tema
  5. Una lista con tres enlaces relacionados con el tema (no es ésta, está más abajo)
  6. El párrafo que describe el contenido de la página (no estaba especificado en la lista)

Y ha de cumplir las siguientes condiciones:

  1. La página ha de ocupar menos de 100Kb (¡OJO con el peso de las imágenes!)
  2. Los ficheros han de estar bien organizados en el repositorio (las imágenes en /img)
  3. Ha de estar bien indentado
  4. No ha de tener faltas de ortografía
  5. EL PROYECTO EJECUTADO DESDE RawGit HA DE SER VALIDADO CORRECTAMENTE DESDE https://validator.w3.org/


EDITORES WEB

Para empezar a crear una página web necesitamos un editor web. Un editor web es el programa o aplicación que utilizamos para construir o crear una página web. En principio nos puede servir cualquier editor de texto (no confundir con un procesador web). Yo he utilizado el bloc de notas de Windows10 para crear el primer archivo html, pero luego lo he subido a mi repositorio para esta práctica en GitHub (LM_PRACTICA_1) y he ido modificando el archivo desde la misma plataforma (en esta página no explicaré nada sobre GitHub). Para otras páginas me descargaré algún editor web más completo. Un editor web es un editor de texto que tiene una serie de funciones que facilitan al diseñador la creación de la página web. Dreamweaver, Sublime Text, Notepad++... son algunos de ellos. En este artículo hay un análisis de Dreamweaver y algunas alternativas a este editor web.



ETIQUETAS Y ATRIBUTOS

Las etiquetas son códigos que están entre los símbolos < y >. La mayoría se han de cerrar con el signo /, aunque hay etiquetas que no se cierran.

Ejemplos:

<head> la cierro con </head>

Los atributos se ponen dentro de las etiquetas.

Por ejemplo, para poner un enlace:

<a href="url del enlace">texto que aparecerá</a>

a es la etiqueta y href el atributo. El atributo se pone en la etiqueta de apertura. Se le asigna un valor poniendo detrás del atributo el signo = y escribiendo entre comillas (") el valor del atributo (en este caso la url del enlace). Después añadimos el texto que aparecerá en la página, y finalmente cerramos la etiqueta.



ETIQUETAS QUE HE UTILIZADO

Etiqueta (Tag) Descripción Atributos
<p> PÁRRAFO align
<br> SALTO DE LÍNEA
<b> BOLD (texto en negrita)

<img> IMAGEN, <img src="path relativo" alt="texto que saldrá en caso de no cargarse la imagen"/> alt, src, etc.


ESTRUCTURA BÁSICA

La estructura básica de un documento html5 es la siguiente:

<!DOCTYPE html>

<html>

  <head>

  </head>

  <body>

  </body>

</html>



BREVE DESCRICIÓN DE LA ESTRUCTURA

<!DOCTYPE html> nos indica que la página web será programada siguiendo el estándar html5(información imprescindible para los navegadores y para programodores que trabajen en nuestra página en el futuro).

<html> entre esta etiqueta y su correspondiente de cierre va el código de nuestra página.

<head> contendrá las etiquetas del <title> (contiene el título que se ve en la pestaña del navegador), y etiquetas meta (como por ejemplo <meta name="description" content="breve descripción de la página que aparecerá en el buscador"/>) y otras etiquetas que contienen información sobre otros elementos de la página.

<body> contiene la información visible de la página.
