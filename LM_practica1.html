EDITORES WEB

Para empezar a crear una página web necesitamos un editor web. Un editor web es el programa o aplicación que utilizamos para construir o crear una página web. En principio nos puede servir cualquier editor de texto (no confundir con un procesador web). Yo he utilizado el bloc de notas de Windows10 para crear el primer archivo html, pero luego lo he subido a mi repositorio para esta práctica en GitHub (PRACTICA-1) y he ido modificando el archivo desde la misma plataforma (en esta página no explicaré nada sobre GitHub). Para otras páginas me descargaré algún editor web. Un editor web es un editor de texto que tiene una serie de funciones que facilitan al diseñador la creación de la página web. Dreamweaver, Sublime Text, Notepad++... son algunos de ellos. En este artículo puedes encontrar un análisis del Dreamweaver y algunas alternativas a este editor web.



ETIQUETAS Y ATRIBUTOS

Las etiquetas son códigos que están entre los símbolos < y >. La mayoría se han de cerrar con el signo /, aunque hay etiquetas que no se han de cerrar.

Ejemplos:

<head> la cierro </head>



ETIQUETAS QUE HE UTILIZADO

Etiqueta (Tag) Descripción Atributos
<p> PÁRRAFO align
<br> SALTO DE LÍNEA
<b> BOLD (texto en negrita)
<i> ITALIC (texto en cursiva)
<ol> LISTA ORDENADA. Se construye anidando la etiqueta <li>elemento de la lista</li> start...
<ul> LISTA DESORDENADA. Se construye anidando la etiqueta <li>elemento de la lista</li>
<table> TABLA. Se construye anidando la etiqueta <tr>(filas), y anidadas en cada fila <th>nombre de la columna</th> para la cabecera y <td>dato</td> para las demás filas
<a> hyperlink, <a href="url del enlace">texto que aparecerá</a> href, hreflang, target, etc.
<img> IMAGEN, <img src="path relativo"> alt, src, etc.


ESTRUCTURA BÁSICA

La estructura básica de un documento html5 es la siguiente:

<!DOCTYPE html>

<html>


<head>

</head>


<body>

</body>


</html>



BREVE DESCRICIÓN DE LA ESTRUCTURA

<!DOCTYPE html> nos indica que la página web será programada siguiendo el estándar html5(información imprescindible para los navegadores y para programodores que trabajen en nuestra página en el futuro).

<html> entre esta etiqueta y su correspondiente de cierre va el código de nuestra página.

<head> contendrá las etiquetas del <title> (contiene el título que se ve en la pestaña del navegador), y etiquetas meta (como por ejemplo <meta name="description" content="breve descripción de la página que aparecerá en el buscador"/>) y otras etiquetas que contienen información sobre otros elementos de la página.

<body> contiene la información visible de la página.
